# Product Images

Place your product images in this directory.

## Image Guidelines

- **Format**: JPG or PNG
- **Recommended size**: 800x800 pixels
- **File naming**: Use descriptive names (e.g., `long-black-wig-1.jpg`)
- **Optimization**: Compress images for web to improve loading speed

## Usage

After adding images, update the `image` property in `js/products-data.js`:

```javascript
image: 'images/products/your-image-name.jpg'
```

Images will automatically display in the product cards.
